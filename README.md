Ai-Powered -Housing Society Managemnet System : 

A mobile-first solution built with React Native Expo and Firebase to digitize and automate residential society management in Pakistan.

Features include automated billing, AI-based complaint classification, real-time notifications, and an intelligent chatbot.

Admins get web dashboards for resident management, analytics, and event tracking.

Designed to improve efficiency, transparency, and user experience for residents and society management.
